# Task 1 – AI Website Copy Generator for Local Businesses

## Business Chosen
**Mokai Café**, Kolhapur, Maharashtra, India

## Project Description
This project demonstrates a structured prompt framework designed to generate conversion-focused, website-ready copy for local businesses using AI.

The goal is to create clear value propositions, service descriptions, and persuasive call-to-action sections that can be directly used on real business websites.

## Deliverables
- Homepage copy
- Services / offerings content
- Call-to-action (CTA) sections

## Tools Used
- ChatGPT

## Skills Demonstrated
- Prompt engineering
- Tone and context control
- Business-focused content generation
- AI workflow documentation
